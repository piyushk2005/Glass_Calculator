# Premium Glass Calculator

## Description
A modern and responsive Glassmorphism Calculator built using HTML, CSS, and JavaScript. This calculator features a premium dark glass UI with smooth animations, keyboard support, and fully functional arithmetic operations for a clean and interactive user experience.

## Features
- Premium Glassmorphism UI Design
- Fully Responsive (Desktop + Mobile)
- Basic Arithmetic Operations (+, -, ×, ÷, %)
- Clear All (AC) Function
- Delete Last Input (DEL)
- Decimal Number Support
- Keyboard Input Support
- Error Handling for Invalid Expressions
- Smooth Hover & Click Animations
- Modern Dark Theme Interface

```bash

```

2. Open project folder:
```bash
cd Glass_Calculator
```

3. Run the project:
Open `index.html` in your browser.
